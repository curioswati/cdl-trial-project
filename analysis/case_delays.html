Now we can plot a corelation between no_proceedings and decision_days.

The general pattern is that as the number of proceedings increase, the time taken in disposal also increase.

But we also see that there are many cases where proceedings were more but disposal was fast.
This hints us to look for other factors that are responsible for delay in disposals.

Let's look at how type of a case affects the disposal time.

So we see, that when the number of proceedings are similar the case types differ, that means some case types are processed faster than others.
But again, we have same case types with similar proceeding count but different disposal times.

Let us see how the case handling Judge affects the time.

So we see, the cases that have same hearings and disposed at different times have different judges most of the time.
Let's look deeper into it.

So it doesn't make much difference, we see the same judge processing the case in different times given same no of proceedings.

Though we also have examples of when a judge does breaks this pattern (at least to some extent).

So we need to look more into the data to find out what affects the disposal time most.
